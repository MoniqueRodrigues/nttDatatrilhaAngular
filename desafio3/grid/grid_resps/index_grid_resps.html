a) Explique o que é o Grid Layout.

Grid layout surgiu para proporcionar um método bimendisional para criação de layout CSS com a possibilidade de layoutar os ítens através do uso de linhas e colunas.

d) O que é a unidade fr?

Unidade fr é uma unidade de medida que representa uma fração do espaço disponível no container do grid.


e) O que é a notação repeat?

É uma maneira reduzida de utilizar quando a definição possue partes repetidas com o objetivo de deixar o código mais simples.

ex: sem repeat():grid-template-columns: 20px [col-start] 20px [col-start] 20px [col-start]; com repeat():grid-template-columns: repeat(3, 20px [col-start]);

f) O que são as linhas do Grid?

Ao ser definido um grid são definidos trilhas do grid e não linhas. O grid devolve linhas numeradas que servem para usarmos para posicionar os itens. Em uma grid de 3 colunas por duas linhas se tem 4 linhas de colunas no total se iniciando da primeira linha da esquerda.


g) O que é uma célula no Grid?

Célula de grid é a união entre linhas e colunas. É a menor unidade de um grid como uma célula de uma tabela. Tendo o grid definido como pai, os itens filhos são organizados cada um em uma célula do grid definido.


h) O que são áreas do Grid?

Áreas do grid é uma ou a junção de mais células em forma de retângulo.
